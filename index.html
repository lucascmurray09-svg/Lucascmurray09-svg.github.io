How this stays safe for everyone.

Built with the same rules a school's own peer-tutoring club would run on.

Verified students only

Every account — tutor or student — is verified with a school email and linked to a staff advisor.

Public, supervised locations only

Sessions happen in the school library or a public library study room during posted open hours. Never off-site, never behind a closed door.

Parent permission on file

A parent or guardian signs a one-time permission form before a student can book or tutor a session.

Easy to report

See something off? Report it in two taps from any profile. A staff advisor reviews every report within a day.
